# IMDB-MOVIES-SQL-
This project focuses on analyzing IMDb movie data using SQL queries to extract meaningful insights from a movie database. The project includes database creation, data retrieval, joins, aggregation, filtering, and advanced SQL operations to answer real-world analytical questions related to movies, directors, ratings, and revenue.
